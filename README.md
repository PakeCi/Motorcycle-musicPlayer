# ESP32 Music Controller

This project using ESP32 to act like a music controller and the best part is **IT HAS A KEYBOARD TO CONTROL THE MUSIC** YEAHHHHHH anyway, it utilized bluetooth keyboard to control next, previous, and pause music from the keyboard.

That preety much it.. but something better cominng on version-2

oh also, it has touchscreen because of 2.4" ILI9341 TFT display, and you can swipe right to next, left to previous, and tap to stop
